
GENERALS
--------

* Simple_stock is a simple stock control software made with codeigniter php framework + jquery.

* All code is "Extreme high quality" wrote by a codeigniter expert.

* All code is fully commented.

* Easy to understand, extend  and add functionalities.

* Easy and small functions.

* Easy jquery implementations, not complex plugins, just a few lines to make it works.


MODELS
------

* Table names are in plural.

* Class names are singular.



CONTROLLERS
-----------

* All controllers has a search_by/(:field_name)/(:value) method that returns a json array of elements matching, this is just for autocompletes.
